## GPIO 引脚分配

### I2C 总线（I2C_NUM_0）

| 信号 | GPIO | 说明 |
|------|------|------|
| SDA | GPIO 1 | I2C 数据线（内部上拉） |
| SCL | GPIO 2 | I2C 时钟线（内部上拉） |

> 挂载设备：QMI8658(0x6A)、PCA9557(0x19)、FT6336(0x38)、ES8311(0x18)、ES7210(0x41)

### LCD 显示屏（SPI3_HOST）

| 信号 | GPIO | 说明 |
|------|------|------|
| MOSI | GPIO 40 | SPI 数据 |
| CLK | GPIO 41 | SPI 时钟（80MHz） |
| DC | GPIO 39 | 数据/命令选择 |
| CS | PCA9557 IO0 | 片选（通过 IO 扩展芯片控制） |
| RST | NC | 复位（不连接，由上电默认复位） |
| BACKLIGHT | GPIO 42 | 背光 PWM 控制（5kHz, 10-bit） |

参数：320×240 @ 16-bit RGB565，SPI mode 2

### 触摸屏（FT6336）

| 信号 | GPIO | 说明 |
|------|------|------|
| SDA | GPIO 1 | I2C 数据（复用 I2C 总线，400kHz） |
| SCL | GPIO 2 | I2C 时钟（复用 I2C 总线） |
| INT | NC | 中断（未使用，采用轮询） |
| RST | NC | 复位（未连接） |

> 注：软件使用 FT5x06 驱动兼容 FT6336。

| VSYNC | GPIO 3 | 帧同步 |

| D0 | GPIO 16 | 数据位 0 |
| D1 | GPIO 18 | 数据位 1 |
| D2 | GPIO 8 | 数据位 2 |
| D3 | GPIO 17 | 数据位 3 |
| D4 | GPIO 15 | 数据位 4 |
| D5 | GPIO 6 | 数据位 5 |
| D6 | GPIO 4 | 数据位 6 |

### I2S 音频

#### 播放通道（ES8311 → NS4150B 功放 → DB1811AB50 扬声器）

| 信号 | GPIO | 说明 |
|------|------|------|
| MCLK | GPIO 38 | 主时钟 (6.144MHz = 384×16kHz) |
| BCK (SCLK) | GPIO 14 | 位时钟 |
| WS (LRCK) | GPIO 13 | 声道选择/帧时钟 |
| DO (DOUT) | GPIO 45 | 音频数据输出（到 DAC） |
| DI (SDIN) | NC | 音频数据输入（播放时不用） |
| PA_EN | PCA9557 IO1 | 功放使能（高电平=开启 NS4150B） |

#### 录音通道（ES7210 ADC ← 双路模拟麦克风）

| 信号 | GPIO | 说明 |
|------|------|------|
| MCLK | GPIO 38 | 主时钟（与播放复用） |
| BCK | GPIO 14 | 位时钟（与播放复用） |
| WS | GPIO 13 | 帧时钟（与播放复用） |
| DI | GPIO 12 | 音频数据输入（从 ADC） |

### MicroSD 卡（SDMMC 1 线模式）

| 信号 | GPIO | 说明 |
|------|------|------|
| CLK | GPIO 47 | SD 时钟 |
| CMD | GPIO 48 | 命令/响应 |
| D0 | GPIO 21 | 数据位 0（1 线模式） |

### 板载按键

| 信号 | GPIO | 说明 |
|------|------|------|
| BOOT (用户键) | GPIO 0 | 用户自定义按键（上拉，按下=低电平） |
| RESET | — | 复位按键（硬件复位，无 GPIO） |

### USB 接口

| 接口 | 控制器 | 说明 |
|------|--------|------|
| USB-UART | CH340K | 串口调试（Type-C 接口） |
| USB OTG | ESP32-S3 内置 | USB 外设/主机（Type-C 接口，用于 TinyUSB HID） |

## PCA9557 IO 扩展器

I2C 地址：**0x19**

| 引脚 | 别名 | 默认状态 | 功能 |
|------|------|---------|------|
| IO0 | LCD_CS | 高电平 (1) | LCD 片选，拉低=选中 |
| IO1 | PA_EN | 低电平 (0) | 音频功放使能，拉高=开启 |
| IO2 | DVP_PWDN | 高电平 (1) | 摄像头电源，拉低=开启 |

- 输出寄存器初始值：`0x05` (IO0=1, IO1=0, IO2=1)
- 方向寄存器：`0xf8` (IO0/1/2 输出, IO3~7 输入)

## 板载硬件模块连接关系

```
┌─────────────────────────────────────────────────────────────────────┐
│                          ESP32-S3-WROOM-1-N16R8                    │
│                                                                     │
│  GPIO1,2 ───── I2C 总线 ─┬── PCA9557 (IO扩展) ── LCD_CS (IO0)     │
│                          │                    ├── PA_EN (IO1) 功放  │
│                          │                    └── DVP_PWDN (IO2)    │
│                          ├── QMI8658 (6轴IMU)                      │
│                          ├── FT6336 (触摸屏)                       │
│                          ├── ES8311 (音频 DAC)                     │
│                          └── ES7210 (音频 ADC) ← 双路模拟麦克风    │
│                                                                     │
│  GPIO40,41,39,42 ──── SPI ──── ST7789 LCD (2.0" 320×240)          │
│                                                                     │
│  GPIO38,14,13,45 ──── I2S TX ──── ES8311 → NS4150B → DB1811AB50   │
│  GPIO38,14,13,12 ──── I2S RX ──── ES7210                          │
│                                                                     │
│  GPIO47,48,21 ────── SDMMC ────── MicroSD 卡槽                    │
│                                                                     │
│  GPIO0 ──────────────── BOOT 按键（用户自定义）                    │
```

## 注意事项

1. **ESP32-S3 USB OTG** 使用板载 Type-C 接口连接电脑，通过 TinyUSB 库实现 HID 设备（键盘/鼠标）。串口通信通过 CH340K 独立输出。
2. **LCD CS** 不直接连接 GPIO，而是通过 PCA9557 IO0 控制，因此驱动初始化前必须先初始化 I2C 和 PCA9557。
3. **摄像头电源** 由 PCA9557 IO2 (DVP_PWDN) 控制：拉低=开启。默认上电为高电平（关闭）。本项目不使用摄像头。
4. **音频功放** 由 PCA9557 IO1 (PA_EN) 控制：拉高=开启 NS4150B。无音频播放时应关闭以省电。
5. **SDMMC** 配置为 1 线模式（仅 D0），可通过修改 `slot_config.width` 切换为 4 线模式。
6. **I2S 播放与录音** 共用 MCLK/BCK/WS 引脚，CODEC 芯片通过不同 I2C 地址区分。handheld 工程中播放使用 I2S_NUM_1，录音示例使用 I2S_NUM_0。
7. **扩展接口** 两个 GH1.25 连接器对外提供 5V/3.3V 电源，信号引脚可通过 GPIO 矩阵配置为 UART/I2C/SPI/PWM/CAN 等。
8. **外壳** 采用无螺丝卡扣设计，可徒手拆装。
