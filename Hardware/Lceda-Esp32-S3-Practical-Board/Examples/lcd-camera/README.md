# 07 - LCD + 摄像头实时显示

立创实战派 ESP32-S3 开发板外设示例 — 将摄像头画面实时显示到 LCD 屏幕。

## 功能

- 先显示一张鹦鹉图片
- 初始化摄像头传感器
- 将摄像头实时画面流式显示到 LCD 屏幕（摄像头预览模式）

## 学习要点

- `bsp_camera_init()` 摄像头初始化
- `app_camera_lcd()` 将摄像头数据帧实时推送到 LCD
- 显示图片 -> 启动摄像头 -> 实时预览的流程控制

## 硬件接线

- 摄像头通过 DVP 接口连接（板载，由 PCA9557 控制电源）

## 编译与烧录

```bash
idf.py set-target esp32s3
idf.py build flash monitor
```

烧录后屏幕先显示鹦鹉图片，随后切换为摄像头实时画面。
